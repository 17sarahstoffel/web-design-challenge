Summary

The purpose of this project was to see if there are any trends on certain weather factors depending on a city's latitude. To do this we took a list of 500 random cites. with Open Weather API to pull data on those 500 cities.


After collecting the data from the Open Weather we used Matplotlib and Jupyter Notebook to create various visualizations on four different weather factors. These included temperature, humidity, cloudiness and wind speed. We compared these factors with the cities latitude.
